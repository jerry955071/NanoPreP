Installation
============

Clone the repository and install with pip:

.. code-block:: bash

   git clone https://github.com/jerry955071/NanoPrePro.git
   cd NanoPrePro
   pip install -e .

Dependencies
------------

NanoPrePro requires:

- Python 3.8+
- Snakemake
- Pandas
- NumPy
- SciPy
- Matplotlib
- Plotly
- Biopython
- minimap2 (for alignment)
