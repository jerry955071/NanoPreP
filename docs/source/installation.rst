Installation
============


**Dependencies**

- Python 3.7+
- Edlib 1.3.8+
- Pandas
- NumPy
- Plotly


Option 1: run with Docker
------------

.. code-block:: bash

   docker run chiaenu/nanoprep:latest nanoprepro --help

Option 2: use pip install
------------

.. code-block:: bash

   pip install nanoprepro
   nanoprepro --help
